Maak in phpmyadmin een database aan genaamd login.

En zet deze code hieronder in de sql tab van je database.

CREATE TABLE users (
	idUsers int(11) AUTO_INCREMENT PRIMARY KEY NOT NULL,
    uidUsers varchar(255) NOT NULL,
    pwdUsers varchar(255) NOT NULL
)

Als je dit hebt ingevoert moet je zelf een username en een password in de table toevoegen.
